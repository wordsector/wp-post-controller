== Description ==

Plugin to control and add more advanced features in posts.

### Features 

* <strong>Post Views</strong>: One can see and display post views.
* <strong>Counter Mode</strong>: Php and Ajax.
* <strong>Custom Post Type Views </strong>: One can select custom post types from the option to enable views.
* <strong>Display Position</strong>: Select position on a post to show counted views to users.
* <strong>Reset Post Views</strong>: Reset post views for all posts or Rest for a particular post to again start fresh views
* <strong>More Features</strong>: Comming soon. Please stay tune to us.
